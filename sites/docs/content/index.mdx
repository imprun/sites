# imprun Platform

imprun은 TypeScript 기반 서버리스 함수를 배포하고 관리하는 플랫폼입니다.

## 주요 기능

- **Cloud Functions**: TypeScript로 작성된 서버리스 함수 배포 및 실행
- **다중 파일 지원**: 여러 모듈로 복잡한 함수를 구조화
- **로컬 개발**: 핫 리로드 기능으로 로컬 테스트
- **스마트 배포**: 개별 함수 또는 변경된 함수만 선택적 배포
- **간편한 CLI**: 사용하기 쉬운 명령줄 도구

## 시작하기

첫 번째 함수를 배포하려면 [빠른 시작](/quick-start)을 참고하세요.

자세한 명령어 참조는 [CLI 가이드](/guides/cli)를 확인하세요.

## CLI 명령어

- `imprun login` - imprun 서버 인증
- `imprun pull` - 기존 프로젝트 다운로드
- `imprun dev` - 로컬 개발 서버 시작
- `imprun build` - 함수 빌드 및 검증
- `imprun deploy` - 서버에 함수 배포
- `imprun templates` - 템플릿으로 함수 생성

전체 문서는 [CLI 가이드](/guides/cli)를 참고하세요.
