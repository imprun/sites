# 빠른 시작

몇 분 안에 imprun을 시작할 수 있습니다.

<Steps>
<Step>
## 설치

imprun CLI를 전역으로 설치합니다:

```bash
npm install -g @imprun/cli
```

설치 확인:

```bash
imprun --version
```

</Step>

<Step>
## 로그인

imprun 서버에 인증합니다:

```bash
imprun login
```

서버 주소를 입력하고 브라우저에서 인증을 완료하세요.

</Step>

<Step>
## 프로젝트 다운로드

기존 프로젝트를 다운로드하거나 새로 생성합니다:

```bash
imprun pull my-app
cd my-app
```

`.imprunrc` 파일과 `functions/` 디렉토리가 포함된 프로젝트 구조가 생성됩니다.

</Step>

<Step>
## 함수 생성

템플릿을 사용하여 첫 번째 함수를 생성합니다:

```bash
imprun templates
```

목록에서 템플릿을 선택하면 `functions/`에 새 함수가 생성됩니다.

</Step>

<Step>
## 로컬 테스트

개발 서버를 시작합니다:

```bash
imprun dev
```

함수를 테스트합니다:

```bash
curl http://localhost:3000/your-function-name
```

</Step>

<Step>
## 배포

함수를 서버에 배포합니다:

```bash
# 전체 함수 배포
imprun deploy

# 또는 특정 함수만 배포
imprun deploy hello

# 또는 변경된 함수만 배포
imprun deploy --changed
```

</Step>
</Steps>

## 다음 단계

- 자세한 명령어 참조는 [CLI 가이드](/guides/cli)를 참고하세요
- `function.yaml`을 사용한 함수 메타데이터 관리 방법 알아보기
- 워크플로우에 맞는 배포 전략 탐색

## 프로젝트 구조

```
my-app/
├── .imprunrc              # 프로젝트 설정
├── functions/             # 함수들
│   └── hello/
│       ├── index.ts       # 함수 코드
│       └── function.yaml  # 메타데이터 (선택사항)
└── .imprun/               # 배포 상태
    └── manifest.json
```

자세한 정보는 [CLI 가이드](/guides/cli)를 참고하세요.
